APPROVEFY – 404 "There's no page at this address" FIX CHECKLIST
===============================================================
Use this when the app shows 404 in Shopify admin when opening the app.

1. SERVER MUST BE REACHABLE
   - After deploy, open: https://approvefy.xloxi.com/health
   - You must see "OK" and status 200. If not, the server is down or URL is wrong.

2. ENV ON SERVER (approvefy.xloxi.com)
   - SHOPIFY_API_KEY = 413942cc5a37489ff893894c4af4ca72 (Client ID from Partners)
   - SHOPIFY_API_SECRET = (Secret from Partners – never commit this)
   - SHOPIFY_APP_URL = https://approvefy.xloxi.com (root URL, no /app at the end)
   - SCOPES = same as in shopify.app.toml (comma-separated)

3. SHOPIFY PARTNERS – APP URL
   - Partners Dashboard → Your app (Approvefy) → Configuration (or App setup)
   - App URL = https://approvefy.xloxi.com (exactly this, no trailing slash)
   - Allowed redirection URL(s) must include:
     https://approvefy.xloxi.com
     https://approvefy.xloxi.com/auth/callback
     https://approvefy.xloxi.com/api/auth
     https://approvefy.xloxi.com/app
   - Save. If you use Shopify CLI, run: shopify app config link (or deploy) so URLs sync from shopify.app.toml.

4. INSTALL APP ON THE STORE
   - 404 often happens if the app is not installed on that store.
   - Partners → Your app → Test your app → Choose store "proxi-embalage" → Install.
   - Then open the app from the store's Apps list (Apps → Approvefy), not by pasting the admin URL.

5. OPEN APP THE RIGHT WAY
   - Do not paste admin.shopify.com/store/.../apps/413942cc5a37489ff893894c4af4ca72 in the browser and expect it to work if the app was never installed.
   - Install the app first (step 4), then click the app name in the store's Apps list.

6. CACHE / BROWSER
   - Try in an incognito/private window.
   - Or clear cache for admin.shopify.com and try again.

If all above are done and 404 persists, check the browser Network tab when opening the app: see if the request to https://approvefy.xloxi.com/?host=... returns 200 or an error.
