12. DEPLOYMENT (Vercel)
--------------------------------------------------------------------------------
- Platform           : Vercel (Node.js serverless)
- Framework preset   : react-router (configured in vercel.json + react-router.config.ts)
- vercel.json:
    framework      = "react-router"
    buildCommand   = "npm run vercel-build"
    installCommand = "npm ci --prefer-offline --no-audit --no-fund --progress=false"
- .vercelignore excludes: extensions, scripts (kept), shopify.app.*.toml,
  .git, .cursor, .gemini, .shopify, .vercel, supabase, build, *.log, etc.
- Postinstall hook   : node scripts/prisma-generate-safe.cjs
- Runtime serve cmd  : prisma migrate deploy && react-router-serve …/index.js

15. NPM SCRIPTS (most useful)
--------------------------------------------------------------------------------
- npm run dev               : local-only dev (no Shopify tunnel)
- npm run dev:shopify       : Shopify CLI dev tunnel
- npm run dev:b2b           : alt config for second test app
- npm run build             : prisma generate + react-router build
- npm run vercel-build      : react-router build only (used by Vercel)
- npm run start             : prisma migrate deploy + react-router-serve
- npm run setup             : prisma generate + migrate deploy
- npm run db:migrate        : prisma migrate deploy
- npm run db:reset          : prisma migrate reset
- npm run lint              : ESLint with cache
- npm run typecheck         : react-router typegen + tsc --noEmit
- npm run config:link       : link Shopify CLI config
- npm run shopify ...       : wraps Shopify CLI
